📊 Exportar para Excel (.xlsx)

Gera uma planilha com todas as abas: Produtos, Estoque, Vendas e Envios. Ideal para análise e compatível com Google Planilhas.

⬇️ Baixar Excel

💾 Backup SQLite (.db)

Arquivo binário do banco SQLite completo. Use para restaurar o banco no servidor ou abrir com DB Browser for SQLite.

⬇️ Baixar .db

📄 Script SQL (.sql)

Exporta todos os dados como comandos INSERT. Pode ser executado em qualquer banco SQLite ou compatível.

⬇️ Baixar .sql

☁️ Como fazer backup no Google Drive

1. Baixe o arquivo .xlsx ou .db acima
2. Acesse drive.google.com
3. Faça upload do arquivo para a sua pasta de backups

💡 Dica: Configure um lembrete semanal para fazer este processo. No Render/Railway o arquivo .db fica salvo na pasta data/ do servidor.
